# 開発ガイド

このディレクトリには、開発者向けの実践的なガイドが格納されています。

## 📄 ファイル一覧

### [setup.md](./setup.md)
開発環境セットアップガイド

**内容**:
- 前提条件（Node.js, Docker等）
- クイックスタート手順
- よく使用するコマンド一覧
- トラブルシューティング
- エディタ設定（VS Code）

**読むべき人**: 全員（最初に実行）

---

### [conventions.md](./conventions.md)
コーディング規約

**内容**:
- TypeScript規約（命名規則、型定義）
- Svelte 5規約（Runes使用）
- レイヤー境界のルール
- テストの書き方
- Git Commit規約
- 禁止事項

**読むべき人**: 全開発者（必読）

---

## 🎯 このディレクトリの目的

「どのように開発するか」を記述した、**実践的な手順書**。

## 📖 読む順序

1. **環境構築**: まず [setup.md](./setup.md) で開発環境をセットアップ
2. **規約確認**: 次に [conventions.md](./conventions.md) でコーディング規約と開発フローを確認
